# TaskManager
## Objetivo
Criar um site simples para gerenciamento de tarefas, permitindo adicionar, visualizar e excluir tarefas.

## Funcionalidades
1. Adicionar tarefas.
2. Visualizar lista de tarefas.
3. Excluir tarefas concluídas.
